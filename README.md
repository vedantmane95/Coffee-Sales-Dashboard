# ☕ Coffee Sales Dashboard (Excel Project)

## 📌 Overview
This project showcases an end-to-end data analysis workflow in Microsoft Excel, where raw coffee sales data is transformed into an interactive dashboard.

The dashboard provides insights into sales performance, customer behavior, and product trends using Excel’s data analysis and visualization capabilities.

---

## 🎯 Objectives
- Clean and transform raw data
- Perform data lookup and integration
- Analyze sales trends and customer behavior
- Build an interactive dashboard for decision-making

---

## 🛠️ Tools & Techniques Used
- Microsoft Excel
- XLOOKUP
- INDEX-MATCH
- IF Functions
- Pivot Tables
- Pivot Charts
- Slicers & Timeline Filters
- Data Cleaning & Formatting

---

## 📂 Dataset Details
The dataset includes:
- Orders data (Order ID, Date, Quantity)
- Customer data (Location, Loyalty Card)
- Product data (Coffee Type, Size, Price)

---

## 📊 Dashboard Features
- 📈 **Sales Over Time (Line Chart)** – Tracks trends by coffee type
- 🌍 **Sales by Country (Bar Chart)** – Compares regional performance
- 🏆 **Top 5 Customers** – Identifies high-value customers
- 🎛️ **Interactive Filters:**
  - Roast Type
  - Package Size
  - Loyalty Card Status
  - Timeline (Date filter)

---

## 🔍 Key Learnings
- Data transformation using XLOOKUP and INDEX-MATCH
- Building dynamic dashboards with Pivot Tables
- Using slicers and timelines for interactivity
- Structuring data for scalable analysis
